# Python Capstone Project: Geodata Visualization

Bu layihə, coğrafi məlumatların `SQLite` bazasından oxunması, `OpenStreetMap API` vasitəsilə emalı və nəticələrin interaktiv xəritədə vizuallaşdırılmasını əhatə edir.

## Proyektin İcrası
*   **Məlumatların emalı**: `geoload.py` proqramı ünvanları API vasitəsilə koordinatlara çevirir və `opengeo.sqlite` bazasında saxlayır[cite: 4].
*   **Vizualizasiya**: `geodump.py` bazadan məlumatları oxuyaraq `where.js` faylını yaradır və `where.html` vasitəsilə xəritədə göstərir[cite: 4].
